## Run Locally

Clone the project

```bash
  git clone https://github.com/programming-hero-web-course-4/b7a11-toy-marketplace-server-side-sh-bd.git
```

Go to the project directory

```bash
  cd b7a11-toy-marketplace-server-side-sh-bd
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  nodemon index.js
```
